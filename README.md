# Braintrust Nexus
## Live version
https://octo-fantastic-engine.vercel.app

## Tech stack
- Next.js
- Supabase
- Tailwind CSS
- Shadcn UI

## Run locally
### 1 Clone
```bash
git clone git@github.com:GoldAndLink/octo-fantastic-engine.git
cd octo-fantastic-engine
```

### 2 Install modules
```bash
pnpm install
```

### 3 Run server
```bash
pnpm run dev
```
